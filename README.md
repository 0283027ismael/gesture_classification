# gesture_classification
Project about gesture classification for deep learning course from Panamerican University Mexico
steps:

1) Get Grabmyo dataset from https://physionet.org/content/grabmyo/1.1.0/
copy Session1,Session2,Session3 folder under current directory

2) run the following script to convert the dataset to images
```
python image_generation.py
```
3) using jupyter run any of the ipynb files
